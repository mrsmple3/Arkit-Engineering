Portable Diagnostic Set (ELPI + EDD)

PDS

The ingenious portable diagnostic set from Akrit Engineering, is the only diagnostic set that integrates both wireless engine pressure indicator (ELPI) and deflection unit (EDD) into one durable.

Order

Our patented technology delivers

Wireless (know-how) solution (up to 10m range), no long cables are required.

Ergonomic, easy to use - fast calculation of data (approximately 5 sec). - easy to read sensor data even on the opposite side of the open crankcase

Utilises ONE industrial mini-computer (iPC) as the engine pressure indicator ELPI and the crankshaft deflection device, EDD.

Fast data collection saves the time - Engine performance takes less than one minute per cylinder. - Engine deflection takes less than one minute per cylinder.

Unique (know-how) TDC finding method. It allows to calculate mean indicated pressure (Pi) and power (Ni) without sensor on the flywheel.

Compact - Total weight with for both units: 3 kg. - Case size 32 x 26 x 7 cm.

1

Connect pressure sensor to the indicator valve.

2

3

Collect wireless data on mini-computer (PDA)

Connect the PDA to PC via USB cable

Import data from a PDA
